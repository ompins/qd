《AI 作品集展示平台》项目说明与使用指南
一、项目概述 
本项目是一个全 AIGC 辅助开发的个人作品集网站，聚焦 AIGC 落地场景展示，集成三大核心模块：AIGC 生成短视频、高考志愿规划 AI 智能体、站内智能问答助手，所有内容从页面设计、脚本撰写到功能实现均有 AI 工具参与，可作为 AI 辅助网页开发的实践参考。
•	线上访问地址：https://ompins.github.io/qd/
•	技术栈：原生 HTML + CSS + JavaScript，无第三方框架依赖，适配 PC / 移动端多端浏览

四、网站使用指南 
1. 基础浏览 
•	顶部导航栏支持平滑滚动跳转：点击「首页/AIGC视频/AI智能体/联系」可直接定位到对应板块
•	移动端自动适配布局，无需手动缩放页面
2. 站内 AI 助手使用 
操作	说明
展开/收起	点击聊天框头部可收起，点击右下角「💬 AI 助手」按钮重新展开
提问示例	✅ 基础问题：「视频是怎么做的？」「智能体能干什么？」「这是作业吗？」
✅ 复杂问题：「2025 高考志愿填报注意事项」「AIGC 有哪些应用场景？」
响应规则	命中内置规则的问题会秒回，其余问题自动调用大模型生成回答
3. 留言反馈 
下滑到「联系我」板块，填写姓名、邮箱、留言内容后提交即可，页面会弹出提交成功提示。
